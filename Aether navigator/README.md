Aether Navigator is an open-source AI orchestration engine focused on deterministic browser-based workflows and controlled automation.

## Vision
To build a user-sovereign execution layer that enables structured, transparent, and reproducible AI-driven task execution.

## Core Principles
- Deterministic execution
- Transparent planning
- User-controlled automation
- Privacy-first architecture

## Architecture Overview
AI → Planner → Deterministic Step Engine → Browser Adapter → Verified Output

See `/docs` for detailed documentation.
